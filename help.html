❓ Frequently Asked Questions

Is ToolNest really free? expand_more

Yes! ToolNest is completely free. All tools run in your browser - no server costs, no hidden fees, no subscriptions. We believe in providing free, accessible tools for everyone.

Are my files and data safe? expand_more

Absolutely. All processing happens locally in your browser. We never upload your files to any server. Your PDFs, text, and personal data never leave your device. Once you close the page, your files are gone from memory.

Do I need to create an account? expand_more

No account needed! Your favorites, settings, and history are saved locally in your browser using localStorage. You can start using tools immediately without any sign-up.

What happens if I clear my browser data? expand_more

Your favorites, settings, and history are stored in your browser's localStorage. If you clear browser data (cache, cookies, site data), they'll be reset. We recommend:

  • Export your data from Dashboard → Settings before clearing browser data
  • Keep a backup of important calculations or citations
  • Use the export feature to save your data
Can I use tools offline? expand_more

Currently, you need an internet connection to load the website. However, once loaded, most tools work offline. We're working on making ToolNest a Progressive Web App (PWA) for full offline functionality.

Why aren't my PDF tools working? expand_more

PDF tools require modern browsers with JavaScript enabled. If they're not working:

  • Update your browser to the latest version
  • Make sure you're not in private/incognito mode (limits localStorage)
  • Check that JavaScript is enabled
  • Try with smaller PDF files (under 50MB)

📘 How-to Guides

📊 Using the GPA Calculator

  1. Select your grading system (Ghana or International)
  2. Click "Add Course" for each class
  3. Enter course name, credit hours, and select grade
  4. Click "Calculate GPA" to see your results
  5. Click "Save This Calculation" to store it
  6. Access saved calculations from your Dashboard

💡 Tip: Your GPA calculations are automatically color-coded (green for excellent, red for failing).

📄 Merging PDFs

  1. Go to PDF Tools and select "Merge PDF"
  2. Drag and drop PDF files or click to browse
  3. Arrange files by dragging them in the list
  4. Click "Merge PDFs" and wait for processing
  5. Download your merged PDF

💡 Tip: You can merge up to 10 files at once. All processing happens in your browser.

✍️ Using the Grammar Checker

  1. Go to Writing Tools and select "Grammar & Spell Checker"
  2. Paste or type your text
  3. Click "Check Grammar"
  4. Review highlighted issues and suggestions
  5. Click "Apply Suggestions" to fix automatically
  6. Copy or download the corrected text

⭐ Saving Favorites

  1. Click the star icon on any tool card
  2. The star will turn yellow when saved
  3. Access all favorites in your Dashboard
  4. Click the star again to remove from favorites

📱 Using on Mobile

  1. ToolNest is fully responsive - works on phones and tablets
  2. Use the mobile menu (☰) to navigate
  3. Tools are optimized for touch screens
  4. For PDF tools, files are processed on your device

🔧 Troubleshooting

Tools not loading?

  • Refresh the page
  • Clear browser cache
  • Disable browser extensions temporarily
  • Try in incognito/private mode
  • Update your browser to latest version

PDF tools not working?

  • Make sure files are valid PDFs (not corrupted)
  • Try with smaller files (under 50MB)
  • Check that JavaScript is enabled
  • Some browsers have PDF limitations in private mode
  • Try a different browser (Chrome, Firefox recommended)

Lost your saved data?

  • Check if you cleared browser history/cache
  • Look for export files you might have saved
  • Check different browser profiles
  • Data cannot be recovered once cleared

⚠️ Always export important data from Dashboard → Settings

ℹ️ About ToolNest

ToolNest is a collection of free, browser-based tools designed for students, writers, and professionals. All tools run entirely in your browser - no files are uploaded to any server.

Our Mission

To provide accessible, privacy-focused tools that help people work more efficiently without compromising their data security.

Technology

  • HTML5, CSS3, Vanilla JavaScript
  • PDF.js and PDF-Lib for PDF processing
  • LocalStorage for user preferences
  • No backend servers - 100% client-side
