# ToolNest - Free Student, PDF & Writing Tools

![ToolNest Logo](assets/icons/icon-192.png)

ToolNest is a collection of free, browser-based tools for students, writers, and anyone working with PDFs. All tools run entirely in your browser - no files are uploaded to any server.

## 🚀 Live Demo

Visit [https://yourusername.github.io/toolnest](https://yourusername.github.io/toolnest)

## ✨ Features

### 📚 Student Tools
- **GPA Calculator** - Calculate your GPA with support for Ghana and international grading systems
- **Pomodoro Timer** - Stay focused with customizable study sessions
- **APA Citation Generator** - Generate citations for books, websites, and journal articles

### 📄 PDF Tools
- **Merge PDF** - Combine multiple PDF files into one
- **Split PDF** - Split PDFs by page ranges or every X pages
- **Reorder Pages** - Rearrange PDF pages with visual thumbnails

### ✍️ Writing Tools
- **Text Summarizer** - Automatically summarize long texts
- **Word Counter** - Count words, characters, sentences, and reading time
- **Essay Outline Builder** - Create structured essay outlines

### 🎯 Additional Features
- 🌙 Dark mode support
- ⭐ Save favorite tools
- 🕘 Recent tools tracking
- 🔍 Global search
- 📱 Fully responsive design
- 💾 All data stays in your browser

## 🛠️ Technology Stack

- HTML5
- CSS3 (with CSS variables for theming)
- Vanilla JavaScript
- PDF.js for PDF processing
- LocalStorage for user preferences
- GitHub Pages for hosting
